# Casting

Se queremos transforma tipo “maior” para
“menor” usamos casting (transformação explícita)
int i = ..;
long n = ..;
n = i; // Válido. i é transformado para long
i = (int) n; // Válido. Pode haver truncamento

# Strings
● Strings são objetos
– Variáveis guardam referências aos objetos
● s1 == s2 é verdadeiro sse s1 e s2 referenciam o mesmo string
● s1.equals(s2) é verdadeiro sse os strings referenciados por s1 e s2 contém
a mesma sequência de caracteres
Concatenando strings com +
Ex: "hola " + " Giovanny" + ", que tal?"

Números para String (se n int e d double)
"" + n
"" + d
String.valueOf(n)
String.valueOf(d)
● String para números
Integer.ParseInt(s)
Double.ParseDouble(s)
